# Модуль `app/public_ai_adapter`

Этот модуль представляет собой сервис-адаптер, который служит мостом между внешними платформами ИИ (такими как Anthropic Claude, OpenAI) и внутренним `MCP-сервером` нашего приложения.

## Назначение

- **Предоставление публичного API:** Создает защищенный публичный эндпоинт, к которому могут обращаться внешние ИИ-модели.
- **Совместимость с `Function Calling` / `Tool Use`:** API спроектировано так, чтобы соответствовать контрактам (форматам запросов и ответов), которые используют ведущие ИИ-платформы для вызова внешних инструментов.
- **"Перевод" запросов:** Адаптер принимает запрос на вызов инструмента от внешней модели, транслирует его в вызов соответствующего метода внутреннего `MCPService`, а затем форматирует ответ в ожидаемый для ИИ-модели вид.
- **Безопасность:** Доступ к эндпоинту защищен с помощью механизма API-ключей.

## Компоненты

- `router.py`: Определяет публичный эндпоинт `/tool_calls`.
- `services.py`: Содержит основную логику трансляции запросов.
- `schemas.py`: Pydantic-схемы, описывающие API-контракт для внешних моделей.
- `security.py`: Реализация проверки API-ключа.
- `dependencies.py`: Зависимости FastAPI для этого модуля.
